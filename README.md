# unity-project-setup-snake

## Overview
This project was created using Unity Engine 6 as part of the project setup task. The goal was to understand how to create a new Unity project and organize it properly.

A simple Snake-style movement system is included to demonstrate basic functionality.


## Template Used
2D Core Template


## Project Structure
The project is organized into the following folders inside the Assets directory:

- **Scenes** – Contains the main game scene  
- **Scripts** – Contains all C# scripts (movement, food, etc.)  
- **Prefabs** – Reusable game objects  
- **Materials** – Materials used in the project  
- **Textures** – Sprites and images  
- **Audio** – Sound effects and music  
- **Animations** – Animation files  

This structure helps keep the project clean and easy to manage.


## Scene Details
The main scene is saved as:

Assets/Scenes/MainScene.unity


## Gameplay (Basic)
- Player (Snake head) moves using arrow keys  
- Simple movement system implemented  
- Optional food interaction included

## How to Run
1. Open the project in Unity Hub  
2. Open `MainScene` from the Scenes folder  
3. Click Play to run the game
